# Merize 功能完整清单 - 基于 Rize 参考

## 总体应用架构

```
[应用启动]
  ├─→ 检查认证状态
  │   ├─→ 已登录 → 进入主界面（侧边栏 + 主内容）
  │   └─→ 未登录 → 进入登录页
  │       ├─→ 输入手机号 → 点击获取验证码
  │       │   ├─→ 成功 → 提示"验证码已发送"
  │       │   └─→ 失败 → 错误信息红色提示
  │       └─→ 输入验证码 → 点击登录
  │           ├─→ 成功 → 跳转到主页
  │           └─→ 失败 → 红色错误提示
  └─→ 完成

[主界面布局]
  左侧固定边栏 (260px) - 所有功能导航
  右侧自适应 - 内容区域
```

## 所有交互响应规则

### 点击
- 所有可点击元素 hover 有视觉变化（背景变深/透明度变化）
- 点击有按压反馈（颜色变化）
- 点击后如果需要等待 → 按钮变成 loading 状态（disabled + 文字"加载中..."）

### 表单
- 输入框 focus → 边框变橙色 `focus:ring-2 focus:ring-[var(--color-accent)]`
- 验证失败 → 红色提示文字

### 错误处理
- API 请求失败 → 弹出错误提示（文字红色）
- UI 乐观更新 → 即使 API 失败，UI 先更新，用户可以再次操作
- console 打印错误详情方便调试

### 持久化
- 所有设置改变立即保存 localStorage
- 数据变更立即保存后端 API
- 页面重新加载恢复所有状态
