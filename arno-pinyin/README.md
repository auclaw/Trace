# Arno 中文简化版 - AI 雅思/托福/多邻国 口语写作批改

Arno 原名 ARNO：AI Reasoning Coach，是一个AI口语写作批改工具。这个是中文简化版，专注中国考鸭需求，更简单更便宜。

## 产品定位

面向**雅思/托福/多邻国考生**，提供平价AI口语写作批改，比人工批改便宜，比免费工具专业，即时出结果不用等。

## 用户痛点

1. 💰 人工批改太贵：一次写作批改 ¥10-20，口语更贵，考生天天练习吃不消
2. ⏰ 人工批改慢：要等几个小时甚至一天，练完早就忘了思路
3. 📱 需要随时能练：吃饭排队都能打开录一段，马上出批改结果
4. 🔓 原版Arno贵，而且全英文对中国考生不友好

## 我们优势

- 💰 **平价**：单次批改 ¥2.99，月卡 ¥29.9 无限次，学生都用得起
- ⚡ **即时**：提交完一分钟内出结果，不用等
- 🇨🇳 **中文友好**：界面全中文，批改结果用中文解释，考生看得懂
- 📱 **多端可用**：网页直接用 + 微信H5，不用下载app

## 启动开发

```bash
# 后端
cd backend
pip install -r requirements.txt
# 复制 config.example.py 为 config.py 填入你的配置
python app.py

# 前端 直接打开index.html就能用
```

## 依赖说明

口语转文字使用 OpenAI Whisper，需要一点点额外依赖：
```bash
pip install openai-whisper
# 需要 ffmpeg
```
