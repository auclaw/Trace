## 使用指南

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/aujingx/betterme.git
cd betterme

# 2. 安装依赖
pnpm install

# 3. 启动应用
pnpm dev

# 4. 打开浏览器
# 访问 http://localhost:3000
```
