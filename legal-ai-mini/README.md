## 快速启动

### 1. 后端部署

```bash
cd backend
pip install -r requirements.txt
# 修改 config.py 填入你的API密钥 (文心一言/豆包，微信小程序，微信支付，数据库)
python app.py
```

**免费部署推荐**：直接部署到 Render，自带免费 PostgreSQL 数据库，起步不用花钱。

### 2. 前端配置

1. 打开微信开发者工具，导入 `frontend` 目录
2. 把所有页面里的 `API_HOST` 改成你的后端地址
3. 提交微信小程序审核，审核通过就能发布
