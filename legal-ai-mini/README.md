# 法律AI小工具 - 微信小程序 MVP

这是法律AI小工具的MVP完整项目框架，包含后端 Flask + 微信小程序前端。

## 项目结构

```
legal-ai-mini/
├── backend/          # Python Flask 后端
│   ├── app.py         # 主程序，三个核心API
│   ├── config.py      # 配置文件，你需要修改这里
│   └── requirements.txt
└── frontend/         # 微信小程序前端
    ├── app.js
    ├── app.json
    ├── pages/
    └── ...
```

## 快速启动

### 1. 后端部署

```bash
cd backend
pip install -r requirements.txt
# 修改 config.py 填入你的API密钥 (文心一言/豆包，微信小程序，微信支付，数据库
python app.py
```

后端需要部署到有公网IP的服务器，可以用阿里云/腾讯云的最低配，一个月几十块钱足够。

### 2. 前端配置

1. 打开微信开发者工具，导入 `frontend` 目录
2. 一键真机调试
3. 在 `app.json 以及各个页面修改 `API_HOST` 改成你的后端地址
3. 提交微信小程序审核，等审核通过就能发布了。
