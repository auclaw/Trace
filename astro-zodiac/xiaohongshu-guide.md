# 小红书星座日运发图指南

## 每天发怎么做

1. **生成文案**：
```bash
cd astro-zodiac
python generate-xiaohongshu.py YYYY-MM-DD
```
就会输出：
- 封面页文案
- 每个星座一页的文案
- 小红书正文文案（带 hashtags）

2. **做图方法（用Canva免费做就行）**：
- 小红书封面尺寸：`1080 × 1440 px`（3:4）
- 做多图：封面 + 12星座一共13张，小红书最多18图，发9图的话可以选前8个星座+封面
- 每个星座一页：星座emoji + 名称 + 运势内容，简单干净好看

3. **模板配色推荐（小红书热门风格）**：
- 浅紫渐变背景 + 白色文字，温柔好看
- 马卡龙色系，年轻人喜欢
- 字体用圆润一点的，比如思源黑体圆角

4. **发的时候文案直接复制生成的正文就行，已经帮你加好了热门标签**

## 文件结构
```
astro-zodiac/
├── content/daily/          # 每日运势markdown文件
├── templates/              # 模板
├── generate-xiaohongshu.py  # 生成小红书图文脚本
└── xiaohongshu-guide.md     # 这份指南
```

## 流量提示
- 结尾一定要引导互动："留下你的星座，接今日好运" 互动率高流量好
- 标签已经帮你选好了：#星座运势 #每日运势 #今日运势 #星座博主 #小红书星座
- 保持日更，粉丝涨得快
