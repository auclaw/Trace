## 一、产品定位
**可颂法务**是面向中国普通人的AI法律文书生成服务，定位：
> 让普通人打得起官司，9.9元起AI生成专业规范的法律文书，支持律师复核

**核心服务范围**: 聚焦普通人小额高频民事法律需求，帮助用户在进入法院/仲裁前准备好符合格式要求的法律文书。

## 二、中国大陆民事诉讼/仲裁完整流程 & 我们的覆盖

### 完整流程 (根据2024修正民事诉讼法，2026现行)

| 阶段 | 流程步骤 | 我们是否覆盖 | 说明 |
|------|----------|--------------|------|
| **准备阶段** | 1. 整理案件事实和证据 | ⭐ 部分覆盖 | 我们通过引导提问/AI对话帮用户梳理信息 |
| | 2. 撰写法律文书 | ✅ **完全覆盖** | 这就是我们的核心产品 |
| | 3. 准备证据材料复印件 | ⭐ 附赠指南 | 我们赠送《证据整理检查清单》，告诉用户怎么整理 |
| | 4. 准备身份证件 + 诉讼费 | ⭐ 附赠指南 | 告诉用户需要带什么，诉讼费怎么算 |
| | 5. 到法院立案/线上申请 | ✖️ 不覆盖 | 需要用户自己操作，我们告诉流程 |
| **受理阶段** | 法院7日内决定是否受理 | ✖️ 法院操作 | 我们告诉用户流程 |
| | 受理后法院5日内发送起诉状给被告 | ✖️ 法院操作 | - |
| | 被告15日内提交答辩状 | ✖️ 法院操作 | 如果被告是用户，我们可以生成答辩状 |
| **审理阶段** | 证据交换 → 开庭审理 → 调解/判决 | ✖️ 法院推进 | 需要用户自己出庭 |
| **执行阶段** | 胜诉后被告不履行 → 申请强制执行 | ⭐ 可扩展 | 若需要我们可以加强制执行申请书模板 |

### 依据
> "如果起诉状 **不符合** 受理标准，法院必须 **7日内裁定不予受理** 并说明原因" — 《民事诉讼法》第85条 (2024修正)

我们的产品定位就是**帮助用户写出符合法院立案标准的起诉状/法律文书**，一次性通过立案，这正是用户需要的。用户不用花几千块律师费只写个文书，我们9.9元搞定。

### 结论
我们精准命中了**用户最痛的点**：普通人不知道怎么写符合格式要求的法律文书，律师费又太贵。我们解决这个问题，后续流程用户自然会走，我们附赠指南告诉用户下一步，覆盖完整够用。

**权威来源**: [Tavily Research - Civil-Law Lawsuit Complaint in China (2026)](https://asianlii.org/cn/legis/cen/laws/cplotproc436/)

- 原告/被告基本信息（姓名/性别/年龄/民族/住址/联系电话）✓
- 明确的诉讼请求 ✓
- 事实与理由 ✓
- **证据清单**（证据名称和来源）✓  *(AI会自动根据案件事实整理)*
- 此致 ××人民法院 ✓
- 具状人签名 日期 ✓
- 需要按被告人数提供同等份数的起诉状副本 ✓  *(产品提示用户)*

**法律依据**: 《中华人民共和国民事诉讼法》第一百二十二条、第一百二十三条 (2024最新修正)
